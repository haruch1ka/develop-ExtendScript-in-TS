## 全体の制作手順（`00中京パイプ_ポルトガル語動画_制作手順.txt` より）

動画仕様: 1920×1080 / H264 / YouTube と同じ書き出し設定。

1. **原稿確認・準備** — Excel の原稿確認・修正（2セリフが1セルに入る／相槌抜け等を元動画を見ながら修正）、左に番号列を作成し上から順に番号を振る。元動画を case ごとに分割（Premiere Pro で手動カット、10本に分割済み）。
2. **音声読み上げ・加工** — セリフを ttsmp3 で音声化 → Audacity で音量調整（ラウドネスノーマライズ -14、MP3 書き出し）。BGM は VideoProc でボーカル除去 → Media Encoder で MP3 化。フォルダ名は Excel の番号と揃える。
3. **セリフPDF（★InDesign／本プロジェクトの対象工程）** — InDesign で文章の画像を作り、After Effects に貼り込んで吹き出しにする。
   - Excel から1つずつコピペし、改行などを揃える。
   - ページ数をセリフの番号（音声と同じ番号）に合わせる。グラフ等セリフ以外の文章も混ざるためページ数に注意。
   - PDF は画質最高・解像度300で書き出し。フォルダ名に注意。
4. **After Effects で動画作成** — 01〜10のフォルダを作り分割動画ごとに作成。動画・BGM・PDF（楕円マスクで吹き出し化）を配置。ポルトガル語は長いため表示タイミングをずらす等で調整。

## 本プロジェクトの対象（確定 2026-07-13）

- 上記**工程3「セリフPDF」の InDesign 作業**を対象とする。現状の手作業（Excel から1つずつコピペ、改行など書式揃え、番号＝ページ合わせ、PDF 書き出し）を ExtendScript で自動化・効率化する。

### 入力元 Excel
